# project-template
Various scripts to create initial project structures based on language

1. Python: setup-project python delphic "Simple project to mimic the oracles"

   Creates the following directory structure:

          delphic/
          ├── .git
          ├── .gitignore
          ├── .venv
          ├── README.md
          ├── delphic
          │   ├── __init__.py
          │   ├── class1.py
          │   ├── main.py
          │   └── module1.py
          ├── setup.cfg
          └── tests
              ├── __init__.py
              ├── test_class1.py
              ├── test_main.py
              └── test_module1.py
